Download

Step 1

Go to the top and download the zip file.

Step 2

Cut the zip file, o to AppData\Local\Programs\Python\Python36\Lib\site-packages in your file explorer and paste the zip file in it.

Step 3

Right click the zip file and press: "Extract All".

Step 4

Go to the __pycache__ folder of the zip file and move the file to the folder of zoro, now move that file what you just moved to the __pycache__ folder of the site-packages folder, and now move the zoro.py file to the site-packages folder.

Step 5

You are done!
